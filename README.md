### Development

**Install Dependencies**

This project is using [lerna](https://github.com/lerna/lerna). Lerna will install dependencies for each of the packages and symlink during postinstall:

```sh
npm install
```

**Local development**

```sh
cd packages/[package-to-work-on]
npm run dev
```
